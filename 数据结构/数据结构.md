## 算法效率度量

### 事前分析估算

算法的好坏

问题的输入规模（和基本操作的数量相关联）

**测定运行时间最可靠的方式就是计算对运行时间有消耗的基本操作的执行次数**

```c
int i, j ,x = 0, sum = 0, n = 100;  // 执行一次
for(i = 1; i <= n; i++)
{
    for(j = 1; j <= n; j++)
    {
        x++;
        sum = sum + x;              // 执行n*n次
    }
}
printf("%d", sum);                  // 执行一次  
```

操作数量 f(n) = n<sup>2</sup>

判断一个算法的效率，常数项和其他次要项往往可以忽略
